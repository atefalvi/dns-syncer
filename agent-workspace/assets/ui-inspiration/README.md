# UI inspiration references

These files are **not brand assets** and are **not final UI designs**.

They are included only to communicate the desired application feel:

- compact app-style layout
- sidebar navigation
- dense operational panels
- clean logs
- local infrastructure control-panel feel

The coding agent must **not copy these screens exactly**.

The final UI must be designed from the PRD using the DataDreamer Observatory design system in:

```text
agent-workspace/assets/reference/datadreamer-observatory-design-system.md
```

Use the inspiration images for layout intent only, not for colors, typography, spacing, or exact component styling.


## Resource note

Do not interpret these references as permission to build a heavy dashboard. The final app must be simple, fast, secure, and resource-efficient.
